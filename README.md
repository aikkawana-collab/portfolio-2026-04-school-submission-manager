# 学校提出物マネージャー

> Google Forms/Sheets/GASを活用し、学校現場の提出物管理を運用コスト0円で自動化するWebアプリ

## 🎯 Google Forms/Sheets/GAS連携で運用コスト0円



## Overview 概要

学校現場における各種提出物（保護者アンケート、健康調査、行事申込など）の収集・進捗管理・督促を一元化するWebアプリです。Google Forms で回収したデータを Google Sheets に蓄積し、Google Apps Script (GAS) を中継するシングルファイル React アプリで可視化・操作します。

サーバーレス・無料枠完結を徹底しており、教育機関で導入しやすいよう「Googleアカウントさえあれば動く」構成にこだわりました。React + TypeScript + Vite + vite-plugin-singlefile により、生成物は単一HTMLファイルとして配布でき、GAS の HtmlService 上で軽量に動作します。


## Problem 何を解決するか

- 紙やメール集約による提出物管理が煩雑で、担当者の業務負荷が大きい
- 未提出者の特定・督促が手作業で、抜け漏れと差戻しが頻発する
- 学校予算の制約上、有償SaaSの新規導入は現実的でなく無料枠で完結させたい

## Solution どうやって解決するか

Google Forms をフロント入口、Sheets をデータベース、GAS を API 層に据え、React + TypeScript で構築したダッシュボードを vite-plugin-singlefile で1ファイル化して GAS の HtmlService から配信します。提出状況・回収率・属性別集計を Recharts で可視化し、未提出者一覧から個別督促までを画面内で完結。Google アカウントの認証をそのまま流用するため、追加のID基盤や課金が不要です。


## Features 主な機能

- Google Formsで集められた提出データを自動取り込みし一覧化する
- クラス・学年・項目別に提出/未提出を可視化する集計ダッシュボード
- 未提出者の絞り込みと一括督促メール文面の生成
- Recharts による回収率推移・属性別グラフの表示
- GAS doGet/doPost をAPI化し、フロントから安全にSheetsを読み書き
- vite-plugin-singlefileで単一HTML化し、GAS HtmlServiceに即デプロイ
