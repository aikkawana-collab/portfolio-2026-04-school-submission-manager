## 1. SLI（Service Level Indicator）定義

| SLI ID | 指標名 | 計測方法 | 単位 |
|--------|--------|---------|------|
| SLI-001 | フォーム→SS反映遅延 | onFormSubmit トリガーの実行時間ログ | 秒 |
| SLI-002 | Webアプリ初期表示時間 | doGet() の実行時間ログ | 秒 |
| SLI-003 | Webアプリ成功率 | 正常レスポンス数 / 総リクエスト数 | % |
| SLI-004 | キャッシュヒット率 | キャッシュヒット数 / 総データ取得数 | % |
| SLI-005 | トリガー成功率 | 成功トリガー数 / 総トリガー発火数 | % |

## 2. SLO（Service Level Objective）定義

| SLO ID | 対応SLI | 目標値 | 計測期間 | 対応要件 |
|--------|---------|--------|---------|---------|
| SLO-001 | SLI-001 | p95 <= 30秒 | 月次 | REQ-NFR-001 |
| SLO-002 | SLI-002 | p95 <= 5秒（キャッシュヒット時）、p95 <= 8秒（キャッシュミス時） | 月次 | REQ-NFR-002 |
| SLO-003 | SLI-003 | >= 99.5%（授業日 8:00-16:00） | 月次 | REQ-NFR-010 |
| SLO-004 | SLI-004 | >= 80% | 週次 | REQ-NFR-003 |
| SLO-005 | SLI-005 | >= 99.9% | 月次 | REQ-FUNC-002 |

## 3. Error Budget Policy

### 3.1 Error Budget 計算

| SLO | 月間許容ダウンタイム/エラー |
|-----|--------------------------|
| SLO-003 (99.5%) | 約22分/月（授業日 8:00-16:00 = 約176時間/月） |

### 3.2 Error Budget 消費時のアクション

| 消費率 | アクション |
|--------|----------|
| 0-50% | 通常運用。新機能開発を継続 |
| 50-80% | 警告。パフォーマンス最適化に注力 |
| 80-100% | 新機能開発を一時停止。信頼性改善に専念 |
| 100%超過 | 緊急対応。原因究明と修正を最優先 |

## 4. 監視・アラート設計

### 4.1 GASスクリプト内の計測実装

```javascript
// 各関数の実行時間を計測・記録するパターン
function measureExecution(functionName, fn) {
  const start = new Date();
  try {
    const result = fn();
    const duration = (new Date() - start) / 1000;
    logMetric(functionName, 'success', duration);
    return result;
  } catch (error) {
    const duration = (new Date() - start) / 1000;
    logMetric(functionName, 'error', duration);
    throw error;
  }
}
```

### 4.2 アラート設定

| アラート | 条件 | 通知先 | 通知方法 |
|---------|------|--------|---------|
| トリガー失敗 | SLI-005 が1回でも失敗 | システム管理教師 | メール |
| 応答遅延 | SLI-002 が連続3回 > 10秒 | システム管理教師 | メール |
| クォータ警告 | 日次APIコールが80%超過 | システム管理教師 | メール |

## 5. SLA（Service Level Agreement）

本システムは学校内部ツールのため、正式なSLAは定義しない。
SLOを内部目標として運用し、Google Workspaceの稼働率に依存することを利用者に周知する。

| 項目 | 内容 |
|------|------|
| 対象時間 | 授業日（平日）8:00-16:00 |
| 対象外 | 休日、長期休暇、Googleサービス障害時 |
| サポート | システム管理教師によるベストエフォート対応 |
