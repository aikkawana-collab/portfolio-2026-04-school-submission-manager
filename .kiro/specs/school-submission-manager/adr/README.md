# Architecture Decision Records (ADR)

本ディレクトリには、プロジェクトの主要な技術的意思決定を記録するADRを格納する。

## ADR テンプレート

新しいADRを作成する際は以下の形式に従うこと。

```markdown
# ADR-XXX: タイトル

## ステータス
提案 / 承認 / 却下 / 廃止

## コンテキスト
意思決定が必要になった背景

## 決定
選択した方針

## 理由
選択の根拠（比較した代替案を含む）

## 結果
この決定による影響（ポジティブ・ネガティブ両方）
```
